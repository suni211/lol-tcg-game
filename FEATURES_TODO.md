# LOL TCG Game - 추가 기능 제안

## 🔥 긴급/필수 기능

### 1. 배틀 로그 상세 보기
**현재 문제:** 배틀 결과만 보이고 실제 진행 과정을 볼 수 없음
**구현 내용:**
- 배틀 히스토리 항목 클릭 시 상세 모달
- 라인전, 미드게임, 팀파이트 단계별 로그 표시
- 누가 어떤 오브젝트를 먹었는지, 킬 스코어 등

### 2. 내 프로필 / 마이페이지
**필요 정보:**
- 전체 전적 (승/패, 승률)
- 티어 히스토리 그래프
- 최다 사용 팀/시즌
- 보유 카드 통계 (전설/영웅/희귀/일반 개수)
- 획득한 업적 (있다면)

### 3. 랭킹 페이지 개선
**현재 상태:** 기본적인 랭킹만 있음
**개선 사항:**
- TOP 100 랭킹 (페이지네이션)
- 티어별 필터링
- 내 순위 하이라이트
- 승률, 총 게임 수, 최근 폼 표시
- 주간/월간 랭킹 변동

---

## 🎮 게임 플레이 개선

### 4. 카드 도감 / 컬렉션
**기능:**
- 전체 카드 목록 (필터: 리그, 포지션, 티어)
- 보유/미보유 표시
- 수집률 (%)
- 각 카드의 상세 스탯 보기
- "도감 완성" 업적

### 5. 출석 보상
**보상:**
- 매일 로그인 시 포인트 지급
- 연속 출석 보너스 (7일, 14일, 30일)
- 출석 체크 캘린더 UI

### 6. 데일리 퀘스트
**예시:**
- "배틀 3회 참여" → 300P
- "승리 1회 달성" → 500P
- "특정 팀 시너지 덱으로 승리" → 1000P
- "카드 5장 구매" → 200P

### 7. 튜토리얼
**필요성:** 신규 유저가 게임을 어떻게 하는지 모를 수 있음
**내용:**
- 회원가입 후 기본 카드팩 무료 지급
- 덱 만드는 법 안내
- 첫 배틀 가이드
- 시너지 설명

---

## 💎 수익화 / 경제 시스템

### 8. 카드 분해 시스템
**기능:**
- 중복 카드 분해 → 포인트 획득
- 티어별 분해 보상 차등
- 일반: 50P, 희귀: 150P, 영웅: 500P, 전설: 2000P

### 9. 카드 거래소 (추후)
**기능:**
- 유저 간 카드 거래
- 경매 시스템
- 시세 그래프

### 10. 프리미엄 화폐 (RP)
**획득 방법:**
- 실제 결제 (추후)
- 이벤트 보상
**사용처:**
- 프리미엄 카드팩
- 특별 스킨/테마
- 닉네임 변경권

---

## 🏆 경쟁 / 소셜

### 11. 친구 시스템
**기능:**
- 친구 추가/삭제
- 친구와 프렌들리 매치 (TP 변동 없음)
- 친구 온라인 상태 표시
- 친구의 덱 구경하기

### 12. 길드/클랜 시스템
**기능:**
- 길드 생성/가입
- 길드전 (주간)
- 길드 채팅
- 길드 기여도 보상

### 13. 시즌 리셋
**기능:**
- 매 시즌(3개월?) 티어 리셋
- 시즌 보상 지급
- 시즌별 리더보드

## 🎨 UI/UX 개선

### 14. 애니메이션 강화
- 카드 뽑기 애니메이션 (뒤집기 효과)
- 배틀 매칭 성공 시 "VS" 화면
- 승리/패배 애니메이션
- 티어 승급 축하 애니메이션

### 15. 사운드 효과
- 버튼 클릭 소리
- 카드 뽑기 소리
- 배틀 시작/종료 사운드
- 승리 팡파레

### 16. 카드 이미지 실제 추가
**현재:** placeholder만 있음
**추가:** 실제 LCK 선수 이미지 (저작권 주의)
- 또는 실루엣/아바타 사용

### 17. 다크 모드
- 라이트/다크 테마 전환
- 사용자 설정 저장
